# 使用ts构建的FP(函数式编程)系统
+ 🚀代码方面解决JS存在的“特性”
+ 🚀提升代码可读性
+ 🚀编程FP化

## How to use
1. Download Deno
+ If you use in deno project, direct use `mod.ts`
+ If you use  `node`/`browser` project, you can use `deno task npm:build` to build this project, and use `./npmBuild/esm/mod`
